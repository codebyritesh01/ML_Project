# 🎓 Student Performance Predictor

A complete End-to-End Machine Learning project that predicts a student's **Math Score** based on academic and demographic factors using Flask and Scikit-learn.

This project demonstrates real-world ML workflow including:

- Data Ingestion
- Data Preprocessing
- Feature Engineering
- Model Training
- Model Evaluation
- Prediction Pipeline
- Flask Web Application
- Logging & Exception Handling
- Modular Project Structure

---

# 📌 Problem Statement

Educational institutions often need insights into student academic performance.

This project predicts a student's **Math Score** using inputs such as:

- Gender
- Ethnicity Group
- Parental Education
- Lunch Type
- Test Preparation Course
- Reading Score
- Writing Score
